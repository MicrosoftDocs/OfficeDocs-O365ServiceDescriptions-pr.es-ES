# <a name="microsoft-dynamics-365-online-service-description"></a>Descripción del servicio Microsoft Dynamics 365 (Online)

A partir de la de 2016 de diciembre de actualización para Dynamics 365 (en línea y local), la funcionalidad de Dynamics CRM es ahora incluidos como parte de Dynamics 365, un conjunto de aplicaciones de negocio inteligente. 365 dinámico unifica la administración de relaciones con los clientes (CRM) y planeación de funciones (ERP) al ofrecer nuevas aplicaciones para ayudar a administrar las funciones de negocio específicos de recurso de empresa. 
  
Se incluyen estas aplicaciones:
  
- Dynamics 365 for Sales
    
- Dynamics 365 for Customer Service
    
- Dynamics 365 for Field Service
    
- Dynamics 365 for Project Service Automation
    
- Dynamics 365 for Operations
    
- Dynamics 365 Financials
    
Para obtener más información sobre las características de Dynamics 365, vea [Microsoft Dynamics 365](https://www.microsoft.com/en-us/dynamics365/home). 
  
## <a name="dynamics-365-plans"></a>Planes de Dynamics 365

Dynamics 365 tiene dos ediciones: Enterprise y profesionales. Dentro de estas dos ediciones, hay disponibles varios planes diseñados específicamente para satisfacer las necesidades de empresas grandes y pequeñas y de los patrones de uso. Las aplicaciones también están disponibles de forma individual. 
  
Para obtener más información sobre las características de Dynamics 365, vea [Microsoft Dynamics 365](https://www.microsoft.com/en-us/dynamics365/home).
