# <a name="sharepoint-online-service-description"></a>Descripción del servicio SharePoint Online

Microsoft SharePoint Online en Office 365 permite a las personas compartir y trabajar juntos, para informar a y contratar a otras personas a través de la empresa, para transformar los procesos de negocio y aprovechar los conocimientos colectivo. Además, SharePoint ofrece capacidades para las organizaciones a proteger y administrar los datos y a crear soluciones personalizadas. Para obtener una introducción rápida, vea [¿Qué es SharePoint?](https://go.microsoft.com/fwlink/p/?LinkId=290976)
  
Para comparar las características de los diferentes planes, consulte [Comparar los planes de Office 365 para empresas](http://go.microsoft.com/fwlink/?LinkID=799177&amp;clcid=0x409).
  
## <a name="whats-new"></a>Novedades

Para obtener más información acerca de las nuevas características en SharePoint Online, consulte [What's new in SharePoint Online](https://go.microsoft.com/fwlink/?LinkId=271744).
  
## <a name="system-requirements"></a>Requisitos del sistema

Para conocer los requisitos del sistema para Office 365, el servicio basado en suscripción mensual disponible para la empresa, educación y organizaciones gubernamentales, consulte [Requisitos del sistema para Office 365](https://products.office.com/office-system-requirements/#Office365forBEG).
  
> [!NOTE]
> Una organización puede implementar un entorno híbrido de SharePoint, que se compone de SharePoint Server 2013 o 2016, normalmente implementado de forma local, y SharePoint Online. Para obtener más información, consulte [Entornos híbridos para SharePoint Server 2013](https://go.microsoft.com/fwlink/p/?LinkId=286111) o [Entornos híbridos para SharePoint Server 2016](https://go.microsoft.com/fwlink/?linkid=847872). 
  
## <a name="sharepoint-online-boundaries-and-limits"></a>Límites de SharePoint Online

Hay algunos límites para el servicio de SharePoint Online administrados por Microsoft, como cuotas de colección de sitios, los límites de carga de archivo y los límites de almacenamiento. Para obtener información detallada, vea [Límites de SharePoint Online](sharepoint-online-limits.md).
  
Para obtener más información acerca de los dispositivos móviles compatibles, consulte [Usar un dispositivo móvil para trabajar con sitios de SharePoint Online](https://go.microsoft.com/fwlink/p/?LinkID=290974).
  
## <a name="subscription-options"></a>Opciones de suscripción

Office 365 está disponible en una variedad de planes a fin de satisfacer las necesidades de su organización de la mejor manera. Para obtener más información sobre distintos planes, incluidas las opciones de los planes independientes e información sobre cómo cambiar de un plan a otro, consulte [Opciones de planes de Office 365](../office-365-platform-service-description/office-365-plan-options.md).
  
Cada persona que tiene acceso a SharePoint Online debe asignarse a un plan de Office 365 o SharePoint Online Plan 1 o 2 de SharePoint Online Plan de servicio independiente. Sin embargo, un plan de Firstline no se vende como una oferta independiente, sólo como parte de la suscripción Firstline, como Firstline de SharePoint Online. Dado que los trabajadores Firstline no pueden ser administradores, los inquilinos con los trabajadores Firstline requieren al menos un administrador de la empresa administrar SharePoint. Los trabajadores Firstline no verán los menús de navegación de SharePoint, por lo que los trabajadores de la Firstline tienen que vaya a los sitios de SharePoint directamente por dirección URL.
  
## <a name="onedrive-for-business"></a>OneDrive para la Empresa

OneDrive para la Empresa (anteriormente SkyDrive Pro) es un servicio de almacenamiento en línea personal para los empleados de una compañía. Los usuarios pueden almacenar, sincronizar y compartir archivos de trabajo en varios dispositivos de manera fácil y segura. OneDrive para la Empresa se incluye en SharePoint Online y también puede adquirirse como un plan independiente. La opción independiente de OneDrive para la Empresa incluye Office Online. Para ver las características que se incluyen en OneDrive para la Empresa con Office Online, vea [Descripción del servicio de OneDrive para la Empresa](../onedrive-for-business-service-description.md).
  
## <a name="sharepoint-online-enterprise-external-users"></a>Usuarios externos de SharePoint Online Enterprise

Los usuarios externos heredan los derechos de uso del cliente de SharePoint Online que les invita a colaborar. Es decir, si una organización compra el plan Enterprise E3 y compila un sitio que usa características de empresa, se concede al usuario externo los derechos de uso o visualización de estas características en la colección de sitios a la que se le ha invitado. Aunque pueden ampliarse los miembros de un proyecto e invitar a usuarios externos a realizar una gran variedad de acciones en un sitio, estos usuarios no dispondrán exactamente de las mismas capacidades que los miembros con licencia completa y de pago de una organización. Obtenga más información acerca de qué [acciones pueden realizar los usuarios externos y cuáles no](https://go.microsoft.com/fwlink/p/?LinkId=397616).
