# <a name="microsoft-365-education"></a>Microsoft 365 Educación

Microsoft 365 está disponible en una variedad de planes para satisfacer mejor las necesidades de su organización. Si busca las diferencias entre los planes de Microsoft 365 y Office 365 Educación, en este artículo se muestran las características que se incluyen en cada uno de ellos.
  
Microsoft 365 proporciona un sistema completo, incluidos Office 365, Windows 10 y Enterprise Mobility and Security. En la tabla siguiente se enumeran las características de Office 365 para educación A1, A3 y A5 junto con las características correspondientes de Microsoft 365 para educación A3 y A5. Para comparar las características de los planes empresariales y empresariales, consulte Comparar los planes de [Microsoft 365](https://go.microsoft.com/fwlink/?linkid=799177)para empresas o, para obtener una lista más detallada de características, consulte la descripción del servicio relevante en las descripciones de servicio de [Microsoft 365 y Office 365.](../office-365-service-descriptions-technet-library.md) Para buscar artículos de soporte técnico e información, consulte [Office Help &amp; Training](https://support.office.com/).
