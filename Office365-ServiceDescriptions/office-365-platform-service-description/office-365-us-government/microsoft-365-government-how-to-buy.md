# <a name="microsoft-365-government---how-to-buy"></a>Microsoft 365 Gobierno - cómo comprar

En respuesta a los requisitos únicos y en evolución del sector público estadounidense, Microsoft ha creado entornos específicos Microsoft 365 gubernamentales para los clientes del sector público estadounidense. Esta descripción del servicio proporciona una visión general de los entornos y más información sobre cómo comprar. Se recomienda leer esta descripción del servicio junto con otras [descripciones de servicio Microsoft 365 y Office 365.](../../office-365-service-descriptions-technet-library.md)
  
## <a name="whats-microsoft-365-government"></a>¿Qué es Microsoft 365 Gobierno?

Microsoft 365 El gobierno es un conjunto de capacidades de software en la nube de productividad, seguridad y movilidad adaptadas a las agencias gubernamentales y contratistas estadounidenses patrocinados para mantener información controlada y no clasificada. Entregado a través de entornos únicos que cumplen con los requisitos de cumplimiento más estrictos, Microsoft 365 Gobierno es una oferta en la nube para clientes del gobierno de EE. UU. que coincide lo más estrechamente posible con las características y capacidades de las ofertas empresariales de la nube comercial de Microsoft.
  
## <a name="whats-the-value-of-microsoft-365-government"></a>¿Cuál es el valor del Gobierno Microsoft 365?

Microsoft 365 El gobierno ayuda a los líderes y trabajadores del gobierno estadounidense a ser productivos en cualquier lugar y en cualquier momento al permitir la productividad móvil y asegurar la colaboración con el cumplimiento ayudando a las agencias a administrar eficazmente los requisitos reglamentarios de cumplimiento de datos. ## <a name="what-makes-microsoft-365-government-different-from-microsoft-365-for-enterprise-offerings"></a>¿Qué hace que Microsoft 365 Gobierno sea diferente de Microsoft 365 para las ofertas empresariales?

Microsoft 365 El gobierno está diseñado exclusivamente para abordar los requisitos de cumplimiento de las agencias gubernamentales estadounidenses en cuatro áreas clave:

**Comunidad exclusiva.** Los centros de datos gubernamentales solo están disponibles para agencias o departamentos gubernamentales, o empresas comerciales que han sido autorizadas a mantener y procesar información controlada en nombre del gobierno de los Estados Unidos.

**Personal examinado.** El acceso al contenido del cliente de su organización está restringido al personal de Microsoft examinado que son ciudadanos estadounidenses y ha pasado las comprobaciones de antecedentes necesarias.
Auditorías de terceros. Toda la infraestructura es auditada por un auditor externo certificado que proporciona un informe de evaluación de seguridad o una carta de certificación utilizada por las agencias federales para emitir una autorización para operar (ATO). 

**Almacenamiento de contenido.** Los datos se almacenan dentro de los Estados Unidos continentales con infraestructura compatible. En el cuadro siguiente se resume Microsoft 365 para los entornos y compromisos gubernamentales.<br><br>

| Ofertas | Compromisos |
|:----------|:----------------------------------|
|Microsoft 365 GCC del Gobierno  <br/> |FedRamp High, DFARS, CJIS, IRS 1075, DISA SRG L2 <br/> |
|Microsoft 365 Gobierno GCC Alto  <br/> |Controles FedRamp High, ITAR, DFARS, DISA SRG L4 <br/> |
|Microsoft 365 Departamento de Gobierno  <br/> |DISA SRG L5  <br/> |

Para obtener más información acerca de la seguridad de Microsoft y la acreditación de cumplimiento, visite el [Centro de confianza de Microsoft ](https://www.microsoft.com/trustcenter/default.aspx)  

## <a name="microsoft-365-government-eligibility-and-validation"></a>Microsoft 365 Elegibilidad y validación del gobierno

Microsoft 365 El gobierno está disponible para los clientes gubernamentales elegibles y las organizaciones no gubernamentales patrocinadas para retener o procesar información controlada. Los criterios de elegibilidad para estas dos categorías de clientes son coherentes en microsoft government cloud.

Tanto GCC como GCC ofertas high están disponibles para cualquier cliente que sea elegible para Microsoft Government Cloud, y el entorno doD es para el uso exclusivo del Departamento de Defensa de ee. UU. La disponibilidad del servicio y el precio difieren, y GCC sigue siendo la oferta de héroe para todos los clientes que no poseen información no clasificada (CUI) controlada por FedRAMP High o DoD.

Un cliente gubernamental elegible se define por uno de los siguientes:

- Una agencia federal, definida como oficina, oficina, agencia, departamento u otra entidad del Gobierno de los Ee. UU.
- Una entidad local/estatal, definida como una de las siguientes:
    - Cualquier agencia de un gobierno estatal o local en los EE.UU.
    - Cualquier condado, municipio, mancomunidad, ciudad, municipio, ciudad, municipio, distrito de propósito especial u otro tipo similar de instrumentalidad gubernamental establecido por las leyes del estado de un cliente y ubicado dentro de la jurisdicción y los límites geográficos del estado del cliente
    - El Distrito de Columbia, la Mancomunidad de Puerto Rico, Guam, Samoa Americana, las Islas Vírgenes de los Estados Unidos y las Islas Marianas del Norte
- Una entidad tribal, definida como una entidad tribal reconocida federalmente elegible para fondos y servicios del Departamento del Interior de los Estados Unidos en virtud de su estatus como tribu india, o, en Alaska, una aldea nativa o una corporación nativa regional de Alaska

Las organizaciones no gubernamentales que poseen los siguientes tipos de datos y pueden proporcionar las pruebas enumeradas califican para el proceso de validación de Microsoft Government Cloud:

Una entidad privada comercial con datos sujetos a normativa. Los tipos de datos gubernamentales aceptados incluyen:
- Tráfico Internacional de Armas (ITAR)
- Información no clasificada controlada (CII)
- Departamento de Defensa (DoD) Información Nuclear Controlada No Clasificada (UCNI)
- Departamento de Energía (DoE) UCNI
- Información de Justicia Penal (CJI)
- Datos de nivel de impacto del Departamento de Defensa
- Otros tipos de datos que requieren Microsoft 365 Gobierno

Una entidad comercial internacional puede calificar, aunque los datos regulados pueden estar obligados a comprar el servicio a través de su filial estadounidense.

Se requerirá una prueba de pertenencia a uno de los grupos mencionados anteriormente para acceder a Microsoft 365 ofertas gubernamentales.
