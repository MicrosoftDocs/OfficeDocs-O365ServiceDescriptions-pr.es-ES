# <a name="excel-online"></a>Excel Online

Excel Online (anteriormente Excel Web App) amplía su experiencia de Microsoft Excel en el explorador web, donde puede trabajar con libros directamente en el sitio Web donde está almacenado el libro. Todos los clientes de Office 365 pueden ver y editar ligeramente los archivos de Office con Office Online.
  
> [!NOTE]
> Los libros que superan 30 megabytes (MB) no pueden verse en Excel Online desde dentro de Microsoft SharePoint Online. Para ver los archivos más grandes, necesitará la aplicación de escritorio de Excel instalada en su equipo. 
  
> [!NOTE]
> Para que pueda comparar ofertas, las características avanzadas que se enumeran en este artículo incluyen las funciones de aplicaciones de escritorio de Excel que solo están disponibles en el conjunto de aplicaciones Office (por ejemplo, Office Professional Plus, Office Standard u Office 365 ProPlus). Para ver una lista de todas las características de Office Online, como Word Online y PowerPoint Online, consulte [Disponibilidad de características en los planes de Office Online](office-online-service-description.md#feature-availability-across-office-online-plans)
  
Para ver qué tipos de archivo son compatibles con Excel Online, vea [Tipos de archivo admitidos para Office Online](office-online-service-description.md#supported-file-types-for-office-online). Para obtener más información acerca de las diferencias entre Excel Online y la aplicación de escritorio de Excel, consulte [Diferencias entre usar un libro en el explorador y en Excel](https://go.microsoft.com/fwlink/?linkid=390875).
  
## <a name="add-hyperlinks"></a>Agregar hipervínculos
<a name="bkmk_Addhyperlinks"> </a>

Agregue un hipervínculo en una celda para insertar vínculos a páginas web o abrir archivos en la Web. Puede escribir la dirección web directamente en la celda o utilizar el comando Insertar hipervínculo.
  
## <a name="advanced-time-filtering-timeline-slicer"></a>Filtro de tiempo avanzado (segmentación de escala de tiempo)
<a name="bkmk_Advancedtimefiltering"> </a>

El control de escala de tiempo permite filtrar datos modelados. Es una forma visual de ver y cambiar un intervalo continuo de fechas y filtrar objetos basados en elementos dinámicos, como tablas dinámicas y gráficos dinámicos. 
  
## <a name="alignment"></a>Alineación
<a name="bkmk_Alignment"> </a>

Utilice los botones de alineación ubicados en la ficha Inicio para cambiar la alineación del texto. 
  
## <a name="apply-conditional-formatting"></a>Aplicar formato condicional
<a name="bkmk_Applyconditionalformatting"> </a>

En Excel Online, el formato condicional puede verse pero no aplicarse. La aplicación de escritorio de Excel permite el uso completo de esta función y otras características avanzadas. Obtenga más información sobre cómo [usar el formato condicional en Excel 2013](https://go.microsoft.com/fwlink/p/?LinkId=282326). 
  
## <a name="apply-data-validation-to-cells"></a>Aplicar validación de datos a celdas
<a name="bkmk_Applydatavalidationtocells"> </a>

No disponible en Excel Online. Las características avanzadas, como la validación de datos, solo están disponibles en la aplicación de escritorio de Excel. En cambio, puede ver e interactuar con datos habilitados para la validación en Excel Online. Obtenga más información sobre cómo [aplicar la validación de datos a celdas en Excel 2013](https://go.microsoft.com/fwlink/p/?LinkId=282328).
  
## <a name="apply-smartrecommended-formatting"></a>Aplicar formato inteligente/recomendado
<a name="bkmk_Applysmartformatting"> </a>

No disponible en Excel Online. En la aplicación de escritorio de Excel 2013, los gráficos recomendados permiten seleccionar de una variedad de gráficos para una mejor presentación de los datos. Obtenga más información sobre [cómo usar las recomendaciones de gráficos en Excel 2013](https://go.microsoft.com/fwlink/p/?LinkId=282335).

<a name="bkmk_Autocomplete"> </a>

## <a name="vba-and-macro-scripting"></a>Scripting de macros y VBA
<a name="bkmk_VBAMacrosscripting"> </a>

La aplicación de escritorio de Microsoft Excel es una herramienta extremadamente sólida para manipular, analizar y presentar datos. En cambio, en algunos casos, a pesar del conjunto enriquecido de características, es posible que a su organización le resulte más sencillo usar el lenguaje de programación Visual Basic para Aplicaciones (VBA) para crear una macro que realice tareas repetitivas y comunes, o para realizar alguna tarea que la interfaz de usuario (IU) aparentemente no consiga abordar. Aunque no puede crear macros con VBA en Excel Online, sí puede abrir y editar hojas de cálculo habilitadas para VBA sin necesidad de quitar (ni dañar) el contenido de VBA del archivo. Obtenga más información sobre la [introducción a macros](https://go.microsoft.com/fwlink/p/?LinkId=271678).

<a name="bkmk_View3dcharts"> </a>

Visualice gráficos 3D en la hoja de cálculo. No puede crear o editar gráficos 3D en Excel Online.
  
## <a name="what-if-analysis-tools"></a>Herramientas de análisis de condiciones
<a name="bkmk_WhatIfAnalysis"> </a>

No disponible en Excel Online. Puede ver los resultados del análisis de hipótesis en Excel Online, como se esperaba. Si quiere usar herramientas de análisis como Búsqueda de objetivo, Tablas de datos, Solver y Series, necesitará la aplicación de escritorio de Excel. Obtenga más información sobre las [diferencias entre el uso de un libro en el explorador y en la aplicación de escritorio de Excel](https://go.microsoft.com/fwlink/p/?LinkId=273900).
