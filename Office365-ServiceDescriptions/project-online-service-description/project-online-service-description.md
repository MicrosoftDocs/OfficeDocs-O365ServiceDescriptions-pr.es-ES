# <a name="project-online-service-description"></a>Descripción del servicio Project Online

Project online es una solución en línea flexible para la administración de carteras de proyectos (PPM) y trabajo cotidiano. Project Online, que se suministra con Office 365, ofrece prácticas capacidades de administración de proyectos para planear, priorizar y administrar proyectos e inversiones de carteras de proyectos prácticamente desde cualquier lugar y con cualquier dispositivo. Project Online lo pueden usar gestores, administradores y visores de carteras, administradores de recursos y proyectos, y jefes y miembros de equipo. 
  
Project Online está disponible en tres planes: Project Online Essentials, Project Online Professional y Project Online Premium.
  
Para comparar las características de los planes, consulte [los planes y precios de Project Online](https://products.office.com/en-us/project/compare-microsoft-project-management-software).
  
Puede exportar, guardar e imprimir páginas en las descripciones de los servicios de Office 365. Obtenga más información sobre cómo [exportar varias páginas](https://techcommunity.microsoft.com/t5/Project-Support-Blog/Project-Online-New-ways-to-work-with-Team-Assignments/ba-p/362001?advanced=false&collapse_discussion=true&q=New%20ways%20to%20work%20with%20Team%20Assignments&search_type=thread).
  
## <a name="project-online-boundaries-and-limits"></a>Límites de Project Online

Hay algunas limitaciones del servicio de Project Online. Para obtener más información, consulte [Límites de software de Project Online](https://support.office.com/en-us/article/Project-Online-software-boundaries-and-limits-5A09DBCE-1E68-4A7B-B099-D5F1B21BA489).
  
## <a name="data-backup"></a>Copias de seguridad de datos

 Project online tiene la misma directiva de copia de seguridad y retención de datos que Office 365. Para obtener más información, consulte [retención de datos, eliminación y destrucción en Office 365](https://docs.microsoft.com/office365/securitycompliance/office-365-data-retention-deletion-and-destruction-overview).
  
## <a name="use-of-sharepoint-online"></a>Uso de SharePoint Online

El servicio de Project Online necesita el uso de SharePoint Online, que se le aprovisionará como parte del servicio de Project Online. Los derechos de la funcionalidad de SharePoint Online que se proporcionan con Project Online Professional o Premium se limitan a almacenar y acceder a datos para admitir Project online.
  
## <a name="licensing-considerations"></a>Consideraciones sobre licencias

### <a name="internal-users"></a>Usuarios internos

Cualquier interacción en un sitio de Project online requiere una licencia de Project online.
  
- Project online Essentials es una licencia de complemento de integrante de equipo para clientes que ya usan Project Online Professional o Project online Premium en su organización.
    
- Para los casos de uso más avanzados, los usuarios deben tener licencias adecuadas. Los jefes de proyecto necesitan como mínimo una licencia de Project Online Professional. Los jefes de cartera de proyectos necesitan una licencia de Project online Premium.
    
> [!IMPORTANT]
>  Project online Essentials es una licencia de *complemento* de integrante de equipo para clientes que tienen Project Online Professional o Project online Premium. 
<br/>Cuando expire la última licencia de Project Online Professional o Project online Premium, las instancias de Project Web App se eliminarán después de 120 días. 
<br/>Para las suscripciones de prueba de Project online, las instancias de Project Web App se eliminarán 30 días después de que finalice el período de prueba. 
  
### <a name="external-users"></a>Usuarios externos

Las licencias de Project online para usuarios externos siguen la misma directiva que la de los usuarios internos. Cualquier interacción en un sitio de Project online requiere una licencia de Project online.
