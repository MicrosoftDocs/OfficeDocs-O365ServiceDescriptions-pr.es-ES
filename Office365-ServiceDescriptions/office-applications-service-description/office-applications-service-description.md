# <a name="office-applications-service-description"></a><span data-ttu-id="5c0ec-103">Descripción del servicio de aplicaciones de Office</span><span class="sxs-lookup"><span data-stu-id="5c0ec-103">Office Applications Service Description</span></span>

<span data-ttu-id="5c0ec-104">Microsoft Office 365 es un servicio de suscripción que se proporciona con la versión más reciente de las aplicaciones de escritorio de Office que ya está familiarizado con, como Word, Excel y PowerPoint.</span><span class="sxs-lookup"><span data-stu-id="5c0ec-104">Microsoft Office 365 is a subscription service that provides you with the latest version of the Office desktop applications that you're already familiar with, such as Word, Excel, and PowerPoint.</span></span>
  
<span data-ttu-id="5c0ec-p101">Aunque Office 365 es un servicio basado en la nube, las aplicaciones de Office no se ejecutan en la nube. En su lugar, los usuarios pueden descargar las aplicaciones de Office desde el portal de Office 365 e instalarlas en sus equipos locales. Microsoft usa la tecnología Hacer clic y ejecutar para que la descarga y la instalación de las aplicaciones de Office resulten rápidas y sencillas. La característica Hacer clic y ejecutar usa la tecnología de virtualización para ejecutar las aplicaciones de Office en un entorno independiente en el equipo local, lo que permite a los usuarios ejecutar las aplicaciones de Office en paralelo con versiones anteriores de Office.</span><span class="sxs-lookup"><span data-stu-id="5c0ec-p101">Even though Office 365 is a cloud-based service, Office applications don't run in the cloud. Instead, users download Office applications from the Office 365 portal and install them on their local computers. Microsoft uses Click-to-Run technology to make the download and installation of Office applications fast and simple. Click-to-Run uses virtualization technology to run Office applications in a self-contained environment on a local computer, which allows users to run Office applications side-by-side with earlier versions of Office.</span></span>
  
<span data-ttu-id="5c0ec-109">Las aplicaciones de Office están disponibles en versiones de 32 y 64 bits.</span><span class="sxs-lookup"><span data-stu-id="5c0ec-109">Office applications are available in 32-bit and 64-bit editions.</span></span>
